# 快捷键
F1 获得帮助  
F2 实现作图窗和文本窗口的切换  
F3 对象捕捉  
F4 三维对象捕捉  
F5 等轴测平面切换  
F6 控制状态行上坐标的显示方式（UCS)  
F7 栅格显示  
F8 正交模式  
F9 栅格捕捉  
F10 极轴模式  
F11 对象捕捉追踪  
F12 动态输入
